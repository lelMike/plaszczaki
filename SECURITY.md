# Security Policy

## Supported Versions

C++
| Version | Supported          |
| ------- | ------------------ |
| >= 17   | :white_check_mark: |
| < 17    | :x:                |

G++
| Version | Supported          |
| ------- | ------------------ |
|   11+   | :white_check_mark: |
|  < 11   | :x:                |

## Reporting a Vulnerability

Updates will be made, as soon as possible if there are any issues reported.
